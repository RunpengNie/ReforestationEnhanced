# Enhanced Reforestation
Civ 5 mod

## About
Enhanced Reforestation is based on gwennog's Reforestation for VP and FramedArchitecture's Reforestation mods.

## Author
Jerry Nie

## How to Use
1. **Download the ZIP file** - Download the mod repository as a ZIP file and extract it to your desired location.
2. **Rename the folder** - Rename the extracted mod folder to `Enhanced Reforestation (v 1)`.
3. **Move to Mods folder** - Move the renamed folder to `C:\Users\[YourUsername]\Documents\My Games\Sid Meier's Civilization 5\MODS`.

After installation, launch Civilization 5 and enable the "Enhanced Reforestation" mod in the mod menu before starting a new game.
